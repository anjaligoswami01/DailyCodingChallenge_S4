# Day 6 — Machine Learning Intuition

## Objective
Understand how machine learning models learn using Google Teachable Machine.

## Tools Used
- Google Teachable Machine

## Project
Built an image classification model using:
- Class 1: Rubik's Cube
- Class 2: Gaming Controller

## What I Did
- Collected image samples for both classes
- Trained the classifier using Teachable Machine
- Tested live predictions using webcam input
- Observed how the model reacts to different inputs

## Observations
1. More training images improved prediction confidence.
2. Clear and consistent images helped the model classify objects accurately.
3. Different lighting and angles sometimes affected prediction accuracy.

## Outcome
Learned how machine learning models depend heavily on data quality and training examples.
