# MVP Scope Decisions

## Excluded Features

### User Authentication
Not required to prove the mentoring value proposition.

### User Accounts
Can be added after validating demand.

### Progress Tracking
Requires long-term user data and is not essential for MVP.

### Resume Review
Expands beyond the primary mentoring use case.

### Interview Simulator
Adds complexity without validating the core AI guidance function.

### Analytics Dashboard
Useful later when real users exist.

### Community Features
Not necessary for initial validation.

### Job Recommendation Engine
Can be introduced after establishing mentoring effectiveness.

### Personalized Learning Plans
Requires user profiles and learning history.

### Multi-Agent System
Single-agent mentoring is sufficient for MVP validation.
