# 📅 Day 3 – NLP Preprocessing

## 📌 Overview

Implemented a basic NLP preprocessing pipeline to convert raw text into machine-readable format.

## ⚙️ Steps Covered

* Tokenization
* Stopword & punctuation removal
* Text normalization (lowercase)
* Bag of Words (BoW) creation
* Conversion to Pandas DataFrame

## 🛠 Tech Stack

Python, NLTK, Pandas

## 🚀 Outcome

Generated a clean, structured dataset ready for machine learning input.
