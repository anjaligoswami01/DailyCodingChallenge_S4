Project Idea

Build a simple semantic similarity system using sentence embeddings.

The program should:

Convert sentences into embeddings
Compare sentence meanings using cosine similarity
Find the most similar sentence from a dataset

- Embeddings improve modern AI systems like search and recommendation engines.
- Cosine similarity is commonly used to compare embeddings.
