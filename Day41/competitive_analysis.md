# Competitive Analysis

## Competitor 1: ChatGPT

Provides general career advice and resume suggestions but lacks structured portfolio-specific evaluation and scoring.

### How Our Product Differs

Combines resume review, GitHub project analysis, portfolio assessment, and personalized career mentoring in one workflow.

---

## Competitor 2: GitHub Copilot

Focuses primarily on coding assistance and code generation.

### How Our Product Differs

Evaluates complete developer portfolios and provides career guidance rather than helping write code.

---

## Competitor 3: LinkedIn Career Tools

Offers profile optimization suggestions and job recommendations.

### How Our Product Differs

Analyzes technical projects, GitHub repositories, resumes, and career readiness together instead of focusing only on LinkedIn profiles.

---

## Unique Value Proposition

A single AI mentor that helps users improve resumes, GitHub repositories, portfolios, and career readiness through personalized recommendations and measurable scoring.
