# AI Movie Retrieval System 🎬

This project demonstrates a semantic retrieval system using sentence embeddings and cosine similarity.

Users can search movie ideas in natural language, and the AI retrieves the most relevant movie plots.

## Features
- Semantic search
- AI-powered retrieval
- Embedding similarity
- Top 3 recommendations

## Tech Stack
- Python
- Sentence Transformers
- Scikit-learn

## Example Queries
- "space survival"
- "dream world"
- "magic powers"
