# Semantic Search Engine 🚀

## Overview
This project implements a semantic search engine using Sentence Transformers.

Unlike traditional keyword search, this system understands the meaning of text using embeddings.

## Features
- Converts text into vector embeddings
- Uses cosine similarity for comparison
- Returns top relevant results

## Tech Stack
- Python
- Sentence Transformers
- Scikit-learn

## How It Works
1. Documents are converted into embeddings
2. Query is converted into embedding
3. Similarity is calculated
4. Top matches are returned

## Example
Query: AI and machine learning

Results:
- Machine learning is a subset of AI
- Deep learning models use neural networks

## Learnings
- Importance of embeddings in AI systems
- Difference between keyword search and semantic search
- Practical implementation of similarity search
